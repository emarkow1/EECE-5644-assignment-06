# EECE-5644-assignment-06

This assignment revolved around training an LSTM model to predict a 24-hour period of home energy use data given a 7 day period of input data. 

## Running the assignment

Dependencied must be installed using 
```
pip install -r requirements.txt
```

The jupyter notebook [LSTM.ipynb](LSTM.ipynb) can then be run from start to finish. It will run much faster using CUDA if available.
